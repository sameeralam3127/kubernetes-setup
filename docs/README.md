# Documentation Guide

This `docs/` folder is the main reading path for people who want to understand the repository before running the setup.

## Recommended Reading Order

1. [macos-setup.md](macos-setup.md)
2. [kubernetes-architecture.md](kubernetes-architecture.md)
3. [container-runtime-stack.md](container-runtime-stack.md)
4. [troubleshooting.md](troubleshooting.md)
5. [../setup-scripts/README.md](../setup-scripts/README.md)

## What Each Area Covers

- `docs/`: concepts, architecture, platform selection, and troubleshooting
- `setup-scripts/`: hands-on cluster bootstrap using Multipass and `kubeadm`
- `k8s/`: manifests for the sample application
- `app/`: FastAPI sample app for Kubernetes practice
